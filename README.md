# Building the Floor First

Source for [my blog](https://rohitchandramouli.github.io) — a weekly record of a six-month foundation sprint into ML engineering, reinforcement learning, and game AI.

I'm an EEE graduate building the technical floor I didn't get from a standard CS pipeline: Python and math fundamentals, PyTorch, transformer architecture, LLM fine-tuning, and a flagship project combining retrieval, agents, and game AI into a working NPC system. This blog is the honest version of that process — including the weeks that didn't go to plan.

Built with Jekyll, hosted on GitHub Pages.
